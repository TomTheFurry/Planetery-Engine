Enum for what action to do for the transparentcy channel

This changes the action that applies when the SwapchainImage is presented to the native OSRenderSurface.

Note
Operating Systems only support SurfaceTransparentAction::RemoveAlpha.

See vulkan docs...

Enumerator

Set alpha channel to 1.0. Supported by most type of OS.

PreMultiplied 

Notify no action is needed as alpha channel is pre-multiplied.

PostMultiplied 

Notify multipying alpha channel is needed.
