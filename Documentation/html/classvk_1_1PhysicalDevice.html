Detailed Description

A physical device.

This is a repesentation of a hardware, like a GPU. It caches the hardware related informations, used by other classes to properly construct an object that is compatible with the underlying hardware

◆ getUsablePhysicalDevice()

PhysicalDevice * PhysicalDevice::getUsablePhysicalDevice ( OSRenderSurface *  osSurface)
static

Get a usable physical device.

Calling this function will run a series of check on all avalible physical devices, and pick the most suitable device. The checks include:

  1. Have default-required and requested Extensions
  2. Have the following Features:
    1. Geometry Shader Support
    2. Tesselation Shader Support
    3. Anisotropic Filtering Support
    4. Todo:
      more to be added

  3. Meet the following Queue family requirements:
    1. At least 1 graphic capable queue family
    2. Bug:
      The first graphic capable queue family supports Present to the optional OSSurface

However, the following properties will effect how suitable a certain device is:

  • Discrete GPU vs Intergrated GPU
  • Max work group size for Compute Shaders
  • Max number of output vertices for Geometry Shaders
  • Max number of output data for Geometry Shaders

  • Max memory size for sparse memories

    Bug:
    This should be total VRAM size...
