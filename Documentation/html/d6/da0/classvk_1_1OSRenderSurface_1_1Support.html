Detailed Description

Support class of OSRenderSurface.

This class is a cache of the support of an OSRenderSurface. It provides access to get/calculate the supported display settings of an OSRenderSurface. calculate*() will get a property that is best matching to the 'perferred' input.

uvec2 OSRenderSurface::Support::calculateImageSize ( uvec2  preferredSize) const

Calculate a best-matching image size based on the inputted 'preferredSize'.

Parameters
preferredSizeThe preferred size of the SwapchainImage, or uvec2(-1) for getting the largest possible size
Returns
A supported SwapchainImage size.
Note
It may return a uvec2(0). In that case, it means that no SwapchainImage should be made, and that making a valid Swapchain is not possible. An example is when the target window is minimized on Windows OS.
