Enumerator
None 

None.

IndirectWritable 

Writable by memory transfer commends.

IndirectReadable 

Readable by commends.

Mappable 

Mappable to host memory, Read/Write control is set via MemoryAccess flag.

Coherent 

Mapped memory is coherent. Any read/write does not have to be flushed.

CoherentMappable 

Equal to Coherent | Mappable
