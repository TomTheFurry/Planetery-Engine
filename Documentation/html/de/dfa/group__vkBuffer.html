Buffer Constructer.

Construct a Buffer using the input parameters.

If 'neededFeature' has MemoryFeature::IndirectWritable, BufferUseType::TransferDst is set.
if 'neededFeature' has MemoryFeature::IndirectReadable, BufferUseType::TransferSrc is set.

Parameters
dThe Logical Device
sizeSize of the Buffer
neededFeatureWhat memory feature is needed for this Buffer
neededUsageHow this Buffer is going to be used
