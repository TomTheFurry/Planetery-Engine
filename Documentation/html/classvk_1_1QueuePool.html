Detailed Description

Queue pool

An object owned by a LogicalDevice. It manages the lifetime and usages of Queues.

Warning
This object should NEVER be constructed manually. Instead, this object is automatically constructed when constructing a LogicalDevice.

In-Use queues mean a queue that have been queried before using queryQueueNotInUse(), and that have yet called markQueueNotUnUse().

Query a CommendPool that is compatible with the queue group. If one with the same CommendPoolType and in the same queue group is not present, it will make a new one.
